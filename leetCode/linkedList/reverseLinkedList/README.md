# LeetCode 75

## Задания по связанным спискам

### 206. Reverse Linked List (Обратный связанный список)

#### Условие

Дан `head` односвязный список, переверните его и верните перевернутый список .

```golang
type ListNode struct {
	Val int
	Next *ListNode
}
```

**Примеры данных**

**Пример 1:**

![example1](img.png)

**Ввод:**

_head_ = [1,2,3,4,5]

**Вывод:** [5,4,3,2,1]

**Пример 2:**

**Ввод:**

![example2](img_1.png)

_head_ = [1,2]

**Вывод:**: [2,1]

**Пример 3:**

**Ввод:**

_head_ = []

**Вывод:**: []
